Seed-ui provides reusable classes to the font size declarations in the type scale and expands the scale with two smaller sizes.

Use the classes from .su_h1 to .su_h6 to give to an element the same scale of the intended heading.

Use the classes .su_small and .su_x-small to give to an element a smaller scale.

Progress Bars

Add the class .su_progress-bar to implement a progress bar element. Then, add the .su_bar class to the child span element. Examples:

To implement the progression add a value between 0% and 100% to the width property of a bar element.

Add one of the following classes to a bar to get some extra colors:

  • Add the class .su_bar--success to get a light green bar
  • Add the class .su_bar--warn to get a light pink bar
  • Add the class .su_bar--error to get a dark pink bar

Add the class .su_progress-bar--rounded to a progress-bar element to make its borders rounded. Examples:

Infinite Loading

Add the class .su_progress-bar--indeterminate to a progress-bar element to make it an indeterminated progress bar. Examples:

Backward

Add the class .su_bar--b to the child span element of the progress-bar element to get a backward animation.

Forward and Backward

Add the class .su_bar--fb to the child span element of the progress-bar element to get a forward and backward animation.

Forms

Add the class .su_form to a form element. Then add each input element inside a container field and give it the class .su_field.

Add the class .su_field-label-up to the field to get the label up when the user click inside the input.

Add the class .su_input to every input element.

Add the class .su_has-error to the field and add an element with class .su_error-message if you want to provide an error message to the end-user.

Important Note

All the attributes of the CSS animations can be manipulated through a data-[CSS animation property] attribute which can be added directly into the HTML element. The system will add for you the prefix of the CSS property automatically (if any).

For Example:

<button class="su_anim--fadeIn" data-animation-duration="5s">Test</button>

<button class="su_anim--fadeIn" data-animation-duration="2s" data-animation-timing-function="ease-out">Test</button>
