Spiral Text Affect

Often on the hover state of a link or button, or even a small body of text, an effect is given to bring attention to the action. The spiralling of text is such an affect that could add a little flare to the hover state.

The effect can be placed on either a <button> or <a> element as well as on any text surrounded by <span> tags. The element gets the evo_c-spiralText class for the effect to take place and each individual text is required to be wrapped by an <em tag.

Buttons and links wil need the __button class appended to have the proper button appearance like so: evo_c-spiralText__button.

Also, if more than one word is used, add the class space to a <span> between each of the words.

Background Size and Shape

The text background on buttons and links can be changed both in shape(circular) and size(full height).

Circular

The background size by default is styled with adequate padding and the shape is rectangular. Yet it can also have a circular appearance.

Full height

The full height background is displayed with the appended classes of --fullHeight or --circlefullHeight. The --circle class is not required if the --circlefullHeight class is present.
