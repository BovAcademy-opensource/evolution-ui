Spacer Classes

A collection of classes that have varying margin values on different sides for quick spacing fixes.

The class .su_space-top--small applies a margin-top: 20px to an element.

The class .su_space-top--medium applies a margin-top: 50px to an element.

The class .su_space-top--large applies a margin-top: 100px to an element.

The class .su_space-right--small applies a margin-right: 20px to an element.

The class .su_space-right--medium applies a margin-right: 50px to an element.

The class .su_space-right--large applies a margin-right: 100px to an element.

The class .su_space-bottom--small applies a margin-bottom: 20px to an element.

The class .su_space-bottom--medium applies a margin-bottom: 50px to an element.

The class .su_space-bottom--large applies a margin-bottom: 100px to an element.

The class .su_space-left--small applies a margin-left: 20px to an element.

The class .su_space-left--medium applies a margin-left: 50px to an element.

The class .su_space-left--large applies a margin-left: 100px to an element.
